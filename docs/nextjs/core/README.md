## Quick Reference

### App Router vs Pages Router
- App Router: Modern approach using `app/` directory (Next.js 13+)
- Pages Router: Legacy approach using `pages/` directory

### Rendering Methods
- **SSG** (Static Site Generation) - Pre-rendered at build time
- **ISR** (Incremental Static Regeneration) - Update static content after build
- **SSR** (Server-Side Rendering) - Rendered on each request
- **CSR** (Client-Side Rendering) - Rendered in the browser
